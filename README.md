# 🌍 Projeto Sapiens

Plataforma educacional inclusiva focada na capacitação e empregabilidade de pessoas com deficiência.

O Projeto Sapiens foi desenvolvido com HTML, CSS e JavaScript, possuindo integração com banco de dados, sistema de login e cadastro, além de recursos avançados de acessibilidade.

Objetivo: Democratizar o acesso à educação e gerar oportunidades profissionais através da tecnologia.

## 🎯 Objetivo do Projeto

Criar uma plataforma digital acessível que una:

- Educação básica e profissionalizante
- Trilhas estruturadas de aprendizado
- Nivelamento educacional
- Inteligência Artificial para apoio aos estudos
- Área de vagas inclusivas

O projeto foi inicialmente estruturado no Miro para organização e com wireframe no Figma, priorizando usabilidade e acessibilidade desde o planejamento.
